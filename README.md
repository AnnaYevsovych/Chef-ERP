## Опис програми

Програма для автоматизації роботи шеф-кухаря та комірника закладу харчування.

Функціонал:
- База рецептур страв з розкладкою інгредієнтів на 1 особу
- Облік продуктів на складі з контролем термінів придатності
- Формування меню на день для заданої кількості персон
- Перевірка достатності запасів на складі
- Формування видаткової накладної та автоматичне коригування залишків

## Технології

- C# / .NET 8.0
- Windows Forms
- JSON (System.Text.Json)

## Запуск

1. Встановити [.NET 8.0 Runtime](https://dotnet.microsoft.com/download)
2. Клонувати репозиторій або завантажити ZIP
3. Відкрити `ChefApp.sln` у Visual Studio 2022
4. Натиснути F5
